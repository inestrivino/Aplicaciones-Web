Gestión de Flota de Vehículos Eléctricos

Plataforma de movilidad sostenible

Bienvenidos a nuestra plataforma de gestión de flota, diseñada para manejar el uso de vehículos eléctricos de la organización.

Beneficios de los vehículos eléctricos

  1. Reducción del 100% en emisiones directas de CO₂
  2. Ahorro del 70% en costes de mantenimiento frente a vehículos de combustión
  3. Acceso a zonas de bajas emisiones y bonificaciones fiscales

Servicios de nuestra plataforma

  • Monitorización en tiempo real de la flota
  • Gestión de puntos de carga
  • Generación de informes de sostenibilidad
  • Alertas automáticas de mantenimiento

Tecnologías utilizadas

HTML
Lenguaje de marcado para estructurar el contenido de la web
CSS
Lenguaje para definir el estilo y presentación de los documentos HTML
JSON
Formato ligero para intercambio de datos entre servidor y cliente

Este proyecto está desarrollado por estudiantes de la Universidad Complutense de Madrid. Para más información, visita nuestra página de contacto.
